## 打包实践 (code-spliting) 

### 主要内容

使用代码分离，将默认的一个构建文件分成多个，解决了构建输出只有一个文件的话，文件体积过大问题。

具体说明请参考：

[代码分离](https://github.com/lvzhenbang/webpack-learning/tree/master/doc/two/code-splitting.md)

### 运行项目

```
yarn add // 安装依赖

npm run dev // 开发模式

npm run build // 发布模式
```
